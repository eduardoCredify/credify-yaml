## Começando

<Steps>
  ### Obtenha suas credenciais
  Entre em contato com a equipe da Credify para receber suas credenciais de acesso à API.

  ### Gere o token de acesso
  Utilize suas credenciais para gerar o token JWT. O token é válido por **24 horas**.

  ```bash
  curl -X POST https://api.credify.com.br/auth \
    -H "Content-Type: application/json" \
    -d '{ "clientId": "SEU_CLIENT_ID", "clientSecret": "SEU_SECRET" }'
  ```

  ### Faça sua primeira consulta
  Inclua o token no header `Authorization` de todas as requisições:

  ```bash
  curl -X POST https://api.credify.com.br/veiculototal \
    -H "Authorization: Bearer SEU_TOKEN" \
    -H "Content-Type: application/json" \
    -d '{ "Consulta": { "IdConsulta": "1", "Placa": "ABC1234" } }'
  ```
</Steps>
